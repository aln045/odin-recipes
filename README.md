# Odin Recipes

Welcome to the README of **odin-recipes**.

This project was created as part of The Odin Project’s HTML Foundations course. The goal was to build a simple recipe website using only basic HTML, while also practicing Git and GitHub workflows.

---

## 🧑‍🍳 What I Built (HTML Skills)

In this project, I practiced and applied fundamental HTML concepts, including:

- Creating a basic webpage structure using HTML boilerplate (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Using headings (`<h1>`, `<h2>`) to structure content
- Adding paragraphs to describe each recipe
- Creating unordered lists (`<ul>`) for ingredients
- Creating ordered lists (`<ol>`) for step-by-step cooking instructions
- Adding images using the `<img>` tag to visually represent each dish
- Linking multiple pages together using anchor tags (`<a>`) to create navigation between recipes and the homepage

This helped me understand how websites are structured and how different HTML elements work together to form a complete page.

---

## 🔧 Git & GitHub Practice

I also used this project to learn and practice version control using Git and GitHub.

Key things I worked on:

- Initializing a local Git repository
- Tracking changes using `git add`
- Saving progress with meaningful commits using `git commit`
- Connecting my local repository to GitHub using a remote origin
- Pushing code to GitHub using `git push`
- Organizing files into folders (such as `/recipes`)
- Fixing issues with untracked files and understanding repository structure

This helped me understand how developers save progress, manage versions of their work, and collaborate using GitHub.
